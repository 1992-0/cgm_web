# ChadGlobal Market

ChadGlobal Market is a general goods supply and trading company website built with React 19 and Tailwind CSS v4.

## Features

- **Home Page**: Hero section, company introduction, and call-to-actions.
- **About Us**: Company mission, vision, and background.
- **Products**: Categorized product listings with filtering.
- **Contact Us**: Contact form and direct WhatsApp integration.
- **Responsive Design**: Optimized for mobile and desktop devices.

## Tech Stack

- **Framework**: React 19
- **Build Tool**: Vite
- **Styling**: Tailwind CSS v4
- **Icons**: Lucide React
- **Routing**: React Router DOM v7

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Start the development server**:
    ```bash
    npm run dev
    ```

3.  **Build for production**:
    ```bash
    npm run build
    ```

## Project Structure

- `src/components`: Reusable UI components (Header, Footer, Button, etc.)
- `src/pages`: Page components (Home, About, Products, Contact)
- `src/layouts`: Layout wrappers
- `src/data`: Static JSON data for products and categories
- `src/lib`: Utility functions
